EPCI is a set of independent posited (and confirmed) divergent, chronic-infection sequences. APCI are sets of simulated sequences in which the same private mutations found in the EPCI are distributed among a set of simulated sequences. The overall number of mutations, number of spike mutations, number of spike RBD mutations, and number of spike RBM mutations is preserved in each APCI dataset, but these mutations were assigned randomly from the list of actual private mutations present in the EPCI dataset. 100 independent APCI datasets were made and tested by running the mutation-pattern detecting (MPD) function. The files in this repository are based on mutation patterns (MPs) among amino acid positions. In the "Results" file, there is a key explaining the meaning of the column names. At the bottom of the screen, click on the second tab to see this key.

The EPCI_vs_APCI_Stats file records major statistics from the EPCI and APCI results and the EPCI/APCI ratio for each result The key for the EPCI_vs_APCI_Stats file is on the same page as the statistics, below the main tables.

The Clades, Pangos, and Megaclade files list the number of sequences in each MP that belong to a given Pango lineage, Nextstrain Clade, or Megaclade.

Megaclades combine related clades. The Megaclades: pre-VOC, Alpha, Beta, Gamma, Delta, BA.1, BA.2, BA.4/5, XBB, BA.2.86, and recombinant. Details for the precise clades included in each Megaclade are listed in a table included in the spreadsheet.

The same analysis was done for specific amino acid substutions. These results can be found in the MP_Substitutions__Main_Results repository.

Two other analyses using stricter parameters for EPCI qualification can be found in the repository called MP_Positions__Sensitivity_Test_Results_v1 and MP_Positions__Sensitivity_Test_Results_v2
